Cube Internal Monitoring Extension

Namespace: fr.liglab.adele.cube.monitoring

Contributions to the Cube Agent

Controllers

  • MonitoringGui: This extension creates a monitoring Gui that shows the local Runtime Model Component Instances.

How to use it?

Just add it to the list of extensions of the Local Agent configuration file:

<extension id="fr.liglab.adele.cube.monitoring"/>
